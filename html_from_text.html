1.0 The Technical Manual for Text_Webber!


     Refer to the text file in Folder (@WEBBER_TEXT)  which created these instructions to see an example of how  Text Code Marks (TCM)  work! The program (Run_Me4WEBBER), hereafter referred to as ‘The Program‘, creates a HTML Framed Web Site from its root folder and sub folders. However, it is still helpful for creating HTML pages and recording Diplomacy Adjudicator results.

 The Text Code Marks , hereafter referred to as TCM, allow you to program your text file‘s essential elements to produce your HTML documents.


    You can place HTML code in your text files as long as the sections enclosed by less than and greater than marks [< code>] are on the same line. Otherwise, the HTML code does not work correctly, often adding ENTITY HTML code.

Finally: you can use programs such as ‘Libre Office‘, ‘Grammarly‘ to turn the HTML documents into PDF documents.

8.0 TEXT CODE MARKS (abbreviation = TCM)


    These marks are your secret weapon for quickly creating HTML Pages without knowing the vast HTML Rule and Tag library! When placed at the end of a line, a TCM tells The Program how to create an HTML Page! They are composed of a colon (:) and a negative or positive sign and a number (:+-1 to :+-9)! This HTML document  html_from_text.txt  in Folder  @WEBBER_TEXT  is an Instruction File and an Example File. 



        9.0 Text Code Mark (TCM) Rules


    1. Generally, you place TCM Marks at the end of a text line in any order. Index TCM (-1,2 and -2) will cause errors when used with nested TCM 8,7,6 or the Line returns TCM (-5,+5). Use TCM 5 after an index (-1,2,-2) to properly display following indented lines.
     2. TCM Marks (+5,-5) are not compatible with TCM Marks (-1,6,2,-2).
     3 TC Marks (+8,+7,+6) require closing TC Marks (-8,-7,-6) and must be nested when used together.
     4. The ‘TAB‘ key, or a blank text line, containing 4 spaces will separate HTML lines with a 4 space spacer line; an empty line containing no spaces will start a new line without a spacer!
     5. Index (-1) can not be indented, a sub index (-2,2) can be indented. An unindented line following a Sub Index  will run into the Index Line. A Sub-Index (-2,2) text line Indented with spaces will indent the Index Line with spaces grouped in fours, to a max of 24.
     6. Link TCM -3,-4 and +4 can be used after a (:) with Index TCM(-1,2,-2). That way you can jump from a Webber page‘s index to the picture or link quickly. The Program will leave an empty line after such links for HTML formatting reasons.
     eg.  {tail.html} ~secondimage.jpg~ HOME}:-4:-3  will move the Intro back into the tail Frame.
     7. Generally TCM Marks can be placed at the end of a text line in any order.


    That‘s it for the general rules, only nine TCM numbers, seven rules and TCM 3,4 syntax to remember or look up!

If after doing this procedure the program reports some not found files then try to find these files and place them in the  @HTML_HOLD_FILES  Folder.
     If Disk Space is not an issue; Cut and Paste the  @DELETE  Folder to  @WEBBER TEXT\@WEBBER_TEXT_BACKUP\ . That way if you wish to bring an old support file back into service, you may still have it in the backup Folder @DELETE .
